Skins Management

Currently the skins Management in Battle of Stalingrad is not implemented in the game. This means custom skins need to be downloaded and added to the correct folder manually.

Luckily taleks, a Developer of 1CGS, created a little tool to handle the task more or less automatically. Following this link you can read through the post in the BoS forums.

The skins can be uploaded using this link:

Skin upload website

The tool can be downloaded using this link:

Skin Downloader

How to use:

  • Download zip, unpack and copy two files to bin/game folder.
  • Run SkinCollection.exe, wait while skin list is downloaded.
  • Double click on skin, there you will see screenshot (may take some time to be downloaded) and install button.
